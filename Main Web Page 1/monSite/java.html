Testez votre connaissance en JAVA!

  1. Qu'affiche System.out.println("1" + "2") ?
    • Réponse : 12
    • Explication : Le + entre 2 chaîne les concatène
  2. Qu'affiche System.out.println(1 + 2) ?
    • Réponse : 3
    • Explication Le + entre 2 entiers les additionne
  3. Qu'affiche System.out.println("1 + 2 = " + 1 + 2) ?
    • Réponse : 1 + 2 = 12
    • Explication : Le premier + concatène (en transformant le 1 en chaîne). Idem pour le second +
  4. Qu'affiche System.out.println("1 + 2 = " + (1 + 2)) ?
    • Réponse : 1 + 2 = 3
    • Explication : Les parenthèses impliquent qu'on fait le second + (addition). Ensuite, le premier + concatène
  5. Si ok est un booléen initialisé à false, que va faire if (ok = true) System.out.println("OK!") ?
    • Réponse : Afficher OK!
    • Explication : la condition est en fait une assignation qui met ok à vrai. Du coup, la condition est vraie.
